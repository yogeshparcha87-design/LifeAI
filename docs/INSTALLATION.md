# Yogesh AI OS - Installation Guide

## 1. Introduction

This guide explains how to set up Yogesh AI OS development environment.

The project is currently under active development.

---

# 2. System Requirements

## Hardware

Minimum:

- 4 GB RAM
- 10 GB free storage
- Internet connection


Recommended:

- 8 GB RAM or higher
- SSD storage

---

# 3. Required Software

## Mobile Development

Install:

- Flutter SDK
- Dart SDK
- Android Studio
- VS Code


## Backend Development

Install:

- Python 3.11+
- pip
- FastAPI


## Development Tools

Install:

- Git
- GitHub Account
