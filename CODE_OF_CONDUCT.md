# LifeAI Code of Conduct

## Our Commitment

We are committed to creating a welcoming, respectful, and inclusive environment for everyone.

All contributors, users, and community members are expected to treat each other with professionalism and respect.

---

## Our Standards

Examples of positive behavior include:

- Be respectful and kind.
- Welcome new contributors.
- Accept constructive feedback.
- Focus on improving the project.
- Help others learn and grow.
- Communicate professionally.

---

## Unacceptable Behavior

The following behaviors are not allowed:

- Harassment or discrimination
- Hate speech
- Personal attacks
- Trolling
- Spam
- Sharing illegal content
- Sharing private information without permission
- Malicious code or security attacks

---

## Contribution Guidelines

Contributors should:

- Follow the project architecture.
- Write clean and readable code.
- Test changes before submitting.
- Keep documentation updated.
- Use meaningful commit messages.

---

## Reporting Issues

If you experience unacceptable behavior, please create a GitHub Issue or contact the project maintainer.

---

## Enforcement

Project maintainers may:

- Remove inappropriate comments.
- Reject harmful contributions.
- Restrict or remove access from contributors who repeatedly violate this Code of Conduct.

---

## Scope

This Code of Conduct applies to:

- GitHub repository
- Discussions
- Pull Requests
- Issues
- Project documentation
- Community communication
